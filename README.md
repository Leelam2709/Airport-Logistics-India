# ✈️ Airport Logistics India

A full-stack logistics web app built from scratch for managing shipments and bookings.

## 🚀 Features
- JWT Authentication (Login / Signup with protected routes)
- Shipment Booking with unique POD number generation (ALI-XXXXX format)
- localStorage-based shipment tracking
- Contact form with MongoDB integration
- Responsive UI built with HTML, CSS & Vanilla JavaScript

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
